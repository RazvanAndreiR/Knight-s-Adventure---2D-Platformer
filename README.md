# Knight's Adventure - 2D Platformer

## Project Overview
This is my **High School Graduation Project** (Atestat), developed in 2024. My goal was to see what I could actually build by putting my mind to it and using the tools available online. It was my first time using Unity and C# and it served as a personal challenge to take a project from an idea to a finished, playable game.

*How I Built It:
Since I was learning as I went, the development process was all about problem-solving and integration
->I used and modified different tutorials available online to create different parts of the game
->The biggest challenge was not writing the code itself, but making all those different parts work together without breaking the game.

*Game Features:
->Core Gameplay: walking, jumping, double jumping, moving platforms, a score and health system and a simple combat system,
->Enemy AI: basic enemies (thieves and archers) that patrol and react to the player. The thieves use their swords to attack and the archers shoot arrows towards the player,
->UI & Menus: a working main menu, HUD for health/score and a Game Over screen,
->Level Design: A complete 2D level made with free assets.

*Controls
->Movement: "A" & "D" for left-right player movement
->Jump and Double Jump: "Space"
->Weapon: "Left-Click"
->Exit to main menu: "Esc"

## Why this is in my portfolio?
This project demonstrates my ability to:
->**Learn on the fly:** Taking a complex engine like Unity and delivering a finished build,
->**Problem Solving:** Navigating third-party code and making it work in a custom environment,
->**Commitment:** Following a project from the initial concept to a final form.
